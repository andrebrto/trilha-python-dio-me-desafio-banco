<h1>Sistema Bancário Simples em Python</h1>
Este é um projeto de exemplo de um sistema bancário simples desenvolvido em Python.
<ul>
  <li>Criar conta bancária: Permite criar uma nova conta bancária com um número único. </li>
  <li> Criar Usuário: Permite criar um usuário novo e cada usuário terá sem que o CPF seja o repetido. </li>
  <li>Depósito: Permite adicionar fundos a uma conta existente.</li>
  <li>Saque: Permite retirar fundos de uma conta existente, desde que haja saldo suficiente.</li>
  <li>Consulta de saldo: Permite verificar o saldo atual de uma conta.</li>
</ul>
 
<h1>Pré-requisitos</h1>
<ul>
  <li>Python 3.x instalado</li>
  <li> Bibliotecas necessárias: Nenhuma além das bibliotecas padrão do Python.</li>
</ul>
